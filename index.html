Recent Project

Here, you'll find a showcase of my recent machine learning projects, highlighting my commitment to developing intelligent systems that are both innovative and impactful.

DestiMate (Bangkit Academy)

I developed "DestiMate," an Android-based travel recommender system using machine learning techniques. This system applies collaborative filtering and content-based filtering to suggest tourist destinations and accommodations tailored to user preferences. The project involved data collection, preprocessing, and the implementation of cosine similarity and bag of words methods for content-based recommendations. The collaborative filtering model was built using TensorFlow and achieved a training error of 0.1162 and a test error of 0.1215, indicating good prediction accuracy.

MovieLens Recommendation System

I developed a movie recommendation system using content-based filtering and neural network algorithms to provide personalized movie suggestions. The system was trained on the MovieLens dataset, which contains millions of movie ratings and genres. The cosine similarity approach recommends movies based on genre and tag similarity, while the neural network learns from user interaction patterns. Evaluation results showed the neural network model achieved a Mean Squared Error (MSE) of 0.15, indicating accurate predictions and improved personalization.

Name Entity Recognition

This project demonstrates the use of Long Short-Term Memory (LSTM) networks for named entity recognition (NER). Achieving an impressive accuracy of 0.9663, the model effectively identifies and classifies entities within text. The high performance underscores the efficacy of LSTMs in handling sequential data and extracting meaningful information.

Multi-Label Face Classification

This research aims to improve multi-label human face classification (age, gender, and race) using CNNs, achieving accuracies of 82.98% for age, 90.36% for gender, and 79.48% for race with the UTKFace dataset. Data augmentation and increased CNN filters enhanced accuracy, though challenges remain for imbalanced categories like "teenager" age and "Indian" or "Others" races.
